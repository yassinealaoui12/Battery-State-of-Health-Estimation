# Data

## Source
The data used in this project comes from the public dataset provided by the
NASA Ames Prognostics Center of Excellence (PCoE).

Download link :
https://www.nasa.gov/content/prognostics-center-of-excellence-data-set

---

## Description

The dataset is composed of 4 batteries : Battery 1, Battery 2, Battery 3 
and Battery 4. Each battery corresponds to a set of complete cycles 
(charge/discharge) measured under specific experimental conditions. 
The dataset contains 39 cycles per battery, for a total of 156 cycles 
across all four batteries.

---

## Variables

The variables used are : Current, Energy, Temperature, Time, VoltageX, 
and Y (Power, target variable).
